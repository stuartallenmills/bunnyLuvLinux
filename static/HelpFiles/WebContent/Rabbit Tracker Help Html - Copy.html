Rabbit Tracker

The Rabbit Tracker tracks all aspects of a rabbit in the shelter, including intake information, current estimated age, Wellness, Treatments, Vet Visits, Friends and Family, and adoption status.

It can also provide the public with an adoptable rabbits page that is generated automatically from the database, as well as an online adoption form that can be used in place of paper or Word Document Forms.

 

Main Screen

The Main Screen displays current BunnyLuv rabbits. The menu display depends on whether the user is logged in or not.

Main Screen Not Logged In

The Initial Screen has a menu with Home and Adoption, and displays all the rabbits currently on file at BunnyLuv. Clicking on a Rabbits name will bring up more information on that bunny. The Main page also has a Name Search Field and Login button, as well as a Print button.

Name Search Field

The name search field has an auto-complete text box. It will only accept rabbits on file.

Login

The login button takes you to the login screen which allows authorized users to login to the full capabilities of the Rabbit Tracker. If you log in as demo with a password of demo the program using a play database. This can be used for trying things out (filling out a fake adoption form, fake deaths, euthanization, vet visits, etc.) without affecting the real database.

Menu

Home

Returns you to the main screen.

Adoption

Adoption provides 3 capabilities:

1)      Find Companion Rabbits allows to find rabbits by age, sex, and whether they have friends or family

2)      The Adoptable page is designed to be integrated into the actual BunnyLuv.org web site. Home on this page takes you to BunnyLuv.org. Tracker must be selected to return to the tracker.

3)      The Adoption request form allows the user to fill out an online adoption request form, where the data is saved to the Tracker database.

Rabbits Display

The default display is all rabbits at BunnyLuv. For each rabbit the rabbits name, description, status, date in, altered status, and estimated age is displayed. The estimated age is calculated using the current date and the estimated age of the rabbit at intake.

Clicking on a rabbits name takes you to the detailed View Rabbit page.

Main Screen Logged In

If logged in as an authorized user the Main Screen has additional menu items and the users name is displayed along with the log out button.

Adoption

When logged in two additional options are available

Track Adoption Requests

Adoption Requests show adoptions currently in process. An Adoption Request is automatically created when someone fills out an on-line adoption application. Adoption Requests can also be created via the New Adoption Request. An adoption request allows you to enter actions, such as spoke with person, scheduled home check for 10/13, Passed Home check. It allows you to add or delete rabbits to the specific request.

Advanced Search

The Advanced Search allows the user to define multiple search parameters, such as intake start and stop dates, and Status Notes contents (such as a location: Jody or North Herd).

New

New allows adding new rabbits , daily reports, vet visits, treatments, wellness reports, or adoption requests. For vet visits, treatments, wellness, and adoption requests a pop-up window requires that you provide a rabbit name. Only rabbits on file will be accepted.

Admin

Admin allows you to list people on file (people that have adopted or are trying to adopt), add vets, add new users, change your password, or delete a user.

Edit

Set Image

Set image allows you to upload an image of the rabbit. If there is already an image it is replaced with the new image.

Add Adoption Info

Add adoption info allows to enter in a sales pitch for adopting the rabbit, as well as any special needs that the rabbit may require that should be disclosed in advance. The adoption rules field is not currently used, but provides a place holder for special cases like all family, some family, all friends

Add Friends and Family

This allows you to identify friends and/or family for the rabbit. The current rabbit, and the added friend/family rabbit are both updated at the same time. If you are editing Sally, and add Bob as a friend, then Sally is automatically added to Bobs friends. The same is done for families if you make Sophie Abbys parent then Abby is automatically added to Sophies offspring.

Edit Intake Information

Allows you to edit the initial intake information or the Status Notes field. The Status Notes field can be used to indicate anything of importance, such as at Sharons, or in Herd.

Status Notes

When the Status Note changes the user must enter the date of the change in the Status Note Date field which is displayed whenever the Note is changed.

New Rabbit

The New Rabbit form is used to enter new rabbits into the system. Rabbit names must be unique there cannot be a rabbit with the same name on file. An error is generated if a rabbit with the same name is found.

 

The Source Type is Shelter or Other. The Source can be anything (such as Found by Merle in Safeway Parking lot), but should correspond to the source type.

 

The estimated Age at intake is used by the system to calculate the estimated age of the rabbit based on current date.

 

The Desc field should be a simple physical description useful in identifying the rabbit

 

The Altered pull down automatically configures itself based on the rabbit sex selected in the Sex field. The Altered Date is optional

 

Status is BunnyLuv (meaning at BunnyLuv), Died, Euthanized, or Adopted. If the status is not BunnyLuv then a Status/Notes change date field is display and the date of the status change must be entered.

 

Status Notes is required and can be anything healthy, put in Herd whatever might be interesting. If there is nothing interesting you can put none

 

Once the information is entered hit submit.

 

Once the rabbit is entered into the system additional information (picture, friends and family, adoption info) can be entered using the View Rabbit Edit option.

 

Vet Visit Form

The Vet Visit form allows you to enter veterinarian treatment for a rabbit. It requires the date, the vet, the procedure, the reason, procedures and notes (which can be none). You must select one of the three options provided in the Altered/Euthanized drop down menu which provides two pre-defined procedures Euthanized and Spayed or Neutered as well as an option for Other. If the rabbit has been altered then Spayed or Neutered is not displayed. Spayed or Neutered is displayed depending on the rabbits sex.

 

Wellness Form

The Wellness form is for documenting check-ups and grooming. It requires the reporting person, date, weight, problems (which can be none or needed grooming ). Treatment and temperature are optional fields.

 

Died Form

Requires a date and notes. Notes can be anything of interest (reason for death, none).

Euthanized Form

This is a Vet Visit form with Euthanized pre-selected as the procedure. It requires that you select a Vet and provide a reason.

Injury/Illness Treatment

This requires a date, a reason, and a treatment. The treatment field can be of any length and height. Treatments are active until a stop date is entered, and can be tracked and updated using the Injury/Illness Tracking Report.
